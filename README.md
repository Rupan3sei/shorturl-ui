🌤️ 云端短链（Cloud ShortURL）

一个基于 **Cloudflare Workers + KV** 的高性能短链接服务，拥有轻量、快速、免服务器的特性。本仓库提供简洁明亮的中文 UI 前端（index.html + result.html），可与任意短链 Worker 后端配合使用。

---

✨ 功能特性

- 🍃 **超轻 UI**：纯 HTML/CSS/JS，无框架依赖  
- 🚀 **极速响应**：依托 Cloudflare 全球网络  
- 🔗 **短链接生成**：支持随机生成 / 自定义 Key  
- ♻️ **可选唯一映射**：相同长链接对应相同短链  
- 🔥 **阅后即焚**（可选）  
- 🔢 **访问次数统计**（可选）  
- 🧹 **管理后台**：创建、查询、删除、查看全部 Key  
- 📱 **移动端完美适配**
